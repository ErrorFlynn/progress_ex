Extended progress widget

progress_ex is a simple extension of the nana::progress class, adding text to the widget and a few color presets. It modifies none of the existing behavior of the base class, and is constructed and used in exactly the same way. By default, the widget looks no different than a regular nana::progress widget.

Text modes

The text that the widget displays depends on which text mode the widget is using. You can change the text mode by calling progress_ex::text_mode() with one of the following arguments:

  • progress_ex::text_modes::percentage

    Displays what percentage progress::value() is out of progress::amount(). Only works if the progress widget is not in "unknown" mode.

  • progress_ex::text_modes::value

    Displays progress::value() out of progress::amount(). This is useful to indicate overall progress when a list of items is being processed, for example. Only works if the progress widget is not in "unknown" mode.

  • progress_ex::text_modes::caption

    Displays the text that you set by calling progress_ex::caption(). This also works if the progress widget is in "unknown" mode, but without text contrast (see below). Note that this is the default text mode, and doesn't need to be set explicitly unless switching to it from a different text mode.

If you want the widget to stop displaying text, just set the text mode to caption and set the caption to an empty string.

Text contrast

By default, the text intersecting with the "full" part of the widget (the progress bar) is displayed with one color, and the text intersecting with the "empty" part (the background) is displayed with a different color. Each of these two colors is meant to contrast with the part of the widget that they intesect with, to improve legibility. They can be changed by calling void text_contrast_colors(nana::color full, nana::color empty).

If this behavior is not desired, you can disable it by calling void text_contrast(bool enable) with a false argument. The regular, "no contrast" text color can be changed by calling void text_color(nana::color).

Color presets for the progress bar

As you probably know, nana::progress allows you to change the progress bar and background colors using the widget scheme. In addition to that, progress_ex provides a few color presets that you can apply by calling color_preset(color_presets preset, double shade = 0).

The first argument can be any of the progress_ex::color_presets enumerated values, which are: { nana, blue, green, orange, violet, brown }.

The second argument modifies the shade of the colors, and should be in the range of -1.0 to 1.0, although effective values are probably from -0.4 to 0.4. Negative values produce darker shades, zero means no change, and positive values produce lighter shades.
